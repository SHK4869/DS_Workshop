Trapping Rainwater Problem

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it is able to trap after raining.

Input

  • Integet T denoting number of test cases
  • For each test case, Integet N denoting the length of the array
  • For each test case, following N, a list of N integers

Output

  • T integers denoting the amount of water trapped for each testcase

Constraints

Example

Input

2
7
1,5,2,3,1,7,2
7
3,0,0,2,0,4

Output

9
10
